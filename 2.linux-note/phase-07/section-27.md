## ⌨️ 1. Terminal Keyboard Shortcuts (মাউস ছাড়া কাজ)

যাঁরা লিনাক্সে প্রো, তাঁরা টার্মিনালে কাজ করার সময় অ্যারো কী (Arrow Keys) বা মাউস খুব কম ব্যবহার করেন।

### কার্সার দ্রুত সরানো (Navigation):

- **Ctrl + A:** লাইনের একেবারে শুরুতে চলে যাওয়া (Home এর মতো)।
- **Ctrl + E:** লাইনের একেবারে শেষে চলে যাওয়া (End এর মতো)।
- **Alt + B:** এক শব্দ (Word) পেছনে যাওয়া।
- **Alt + F:** এক শব্দ (Word) সামনে যাওয়া।

### দ্রুত মোছা ও কাট-পেস্ট (Yanking):

- **Ctrl + U:** কার্সারের অবস্থান থেকে লাইনের শুরু পর্যন্ত সব মুছে ফেলা (কাট করা)।
- **Ctrl + K:** কার্সারের অবস্থান থেকে লাইনের শেষ পর্যন্ত সব মুছে ফেলা (কাট করা)।
- **Ctrl + Y:** আগে মোছা (U বা K দিয়ে) লেখাটিকে পেস্ট (Yank) করা।
- **Ctrl + W:** কার্সারের ঠিক পেছনের একটি শব্দ মুছে ফেলা।
- **Ctrl + L:** স্ক্রিন ক্লিয়ার করা (কমান্ড `clear` এর শর্টকাট)।

## 🕒 2. Time-saving Tricks (কমান্ড রিকভারি)

### `sudo` দিতে ভুলে গেছেন? (`!!`)

অনেক সময় লম্বা একটি কমান্ড লেখার পর Enter দিলে এরর আসে: `Permission denied`। আবার কমান্ডটি না লিখে শুধু `!!` ব্যবহার করুন:

```bash
apt update
# Error: Permission denied

sudo !!
# এটি নিজে থেকেই `sudo apt update` হিসেবে রান করবে!
```

### The Reverse Search (`Ctrl + R`)

অনেক দিন আগে একটি বিশাল `docker pull` বা `tar` কমান্ড দিয়েছিলেন, এখন মনে পড়ছে না?

1. টার্মিনালে **Ctrl + R** চাপুন।
2. কমান্ডের যেকোনো একটি শব্দ (যেমন `docker` বা `backup`) টাইপ করতে থাকুন।
3. লিনাক্স নিজ থেকেই হিস্ট্রি ঘেঁটে পুরো কমান্ডটি বের করে আনবে।
4. `Enter` চাপলে সেটি রান হবে, অথবা `Right Arrow` চাপলে সেটি এডিট করার অপশন দেবে।

## 🔗 3. Chaining Commands (একাধিক কাজ একসাথে)

প্রতিটি কমান্ডের পর এরর চেক করে পরের কমান্ড দেওয়ার চেয়ে, লজিক দিয়ে কমান্ড যুক্ত করে দিন।

### `&&` (AND Operator) — "যদি প্রথমটি সফল হয়, তবেই দ্বিতীয়টি চালাও"

```bash
# গিট থেকে কোড নামাবে এবং সফল হলে ফোল্ডারে ঢুকবে
git clone https://github.com/the-king/repo.git && cd repo

# সোর্স কোড বিল্ড করবে
./configure && make && sudo make install
```

### `||` (OR Operator) — "যদি প্রথমটি ফেইল করে, তবেই দ্বিতীয়টি চালাও"

```bash
# ফোল্ডার তৈরি করার চেষ্টা করবে, না পারলে এরর মেসেজ প্রিন্ট করবে
mkdir /root/secured || echo "Sorry, you need to be root!"
```

### `;` (Semicolon) — "প্রথমটি সফল হোক বা ফেইল করুক, পরেরটি চালাও"

```bash
# ঘুমাবে, তারপর মেসেজ দেবে (স্লিপ ফেইল করার চান্স নেই)
sleep 5 ; echo "I am awake!"
```

## 🏃 4. Background Processing (টার্মিনাল ফ্রি রাখা)

ধরুন আপনি একটি বিশাল ফাইল ডাউনলোড বা স্ক্রিপ্ট রান করিয়েছেন। এটি শেষ না হওয়া পর্যন্ত ঐ টার্মিনাল উইন্ডোতে আপনি আর কোনো কাজ করতে পারবেন না।

### `&` (Ampersand) দিয়ে পিছনে পাঠানো:

কমান্ডের শেষে শুধু একটি `&` বসিয়ে দিন।

```bash
# সিস্টেমের একটি মস্ত বড় ব্যাকআপ নেওয়ার সময়
tar -czvf full-backup.tar.gz /var/www/html > backup.log &

# আউটপুট: [1] 3452 (এখানে 3452 হলো Process ID বা PID)
# এখন আপনি নিশ্চিন্তে টার্মিনালে অন্য কাজ করতে পারেন।
```

### `nohup` (No Hangup) — "টার্মিনাল কেটে দিলেও চলবে"

`&` দিয়ে কাজ চালানোর পর যদি আপনি টার্মিনাল (বা SSH কানেকশন) কেটে দেন, তবে ব্যাকগ্রাউন্ড প্রসেসটিও বন্ধ হয়ে যাবে। इसे আটকাতে `nohup` ব্যবহৃত হয়।

```bash
# সার্ভারে ঢুকে বিশাল কোনো ডাউনলোড শুরু করে ল্যাপটপ বন্ধ করে দিলে:
nohup wget https://example.com/massive-dataset.zip &

# এর আউটপুট নিজে থেকেই 'nohup.out' নামের একটি ফাইলে জমা হতে থাকবে।
```

## 📥 5. Terminal Downloading (ব্রাউজার ছাড়া ফাইল আনা)

সার্ভারে বা টার্মিনালে তো আর Google Chrome থাকে না!

### `wget` (Web Get)

সবচেয়ে বেসিক ডাউনলোড ম্যানেজার:

```bash
# সিম্পল ডাউনলোড
wget https://releases.ubuntu.com/24.04/ubuntu-24.04-desktop-amd64.iso

# অন্য নামে সেভ করতে (-O)
wget -O my-os.iso https://releases.ubuntu.com/.../
```

### `yt-dlp` (ভিডিও ডাউনলোডার)

এটি টার্মিনালের সেরা ইউটিউব (এবং অন্যান্য সাইট) ডাউনলোডার।

```bash
# ইন্সটল করা (অফিসিয়াল গাব রিলিজ ব্যবহার করা ভালো, তবে apt-এও পাওয়া যায়)
sudo apt install yt-dlp -y

# সাধারণ ভিডিও ডাউনলোড:
yt-dlp "https://www.youtube.com/watch?v=XXXXXXXX"

# শুধু অডিও (mp3) ডাউনলোড করা:
yt-dlp -x --audio-format mp3 "https://www.youtube.com/watch?v=XXXXXXXX"
```

## 🏋️ Practice Exercise (Daily Habits)

### অনুশীলন ১: Keyboard Shortcuts প্র্যাকটিস (মাউস ধরবেন না!)

```bash
# 1. intentionally (ইচ্ছাকৃতভাবে) একটি ভুল কমান্ড লিখুন:
sduo atp udpate

# 2. Ctrl+A চেপে শুরুতে যান। 'sduo' কে 'sudo' করুন (Right arrow এবং ডিলিট দিয়ে)।
# 3. Ctrl+E চেপে শেষে যান এবং Enter দিন।
# 4. 'command not found' এরর আসবে।
# 5. Up Arrow চাপুন, 'atp udpate' কে কাট করতে Ctrl+W ব্যবহার করুন।
```

### অনুশীলন ২: Chaining & Backgrounding

```bash
# 1. একটি ফোল্ডার তৈরি করুন এবং তার ভেতর একটি ফাইল বানিয়ে ব্যাকগ্রাউন্ডে ১০ সেকেন্ড ঘুমান:
mkdir mytest && cd mytest && touch hello.txt && sleep 10 &

# 2. এই ১০ সেকেন্ডের মধ্যে ফাইলটি আছে কিনা চেক করুন (টার্মিনাল কিন্তু ফ্রি আছে!):
ls -l
```
