## ☕ 1. The Morning Routine (সিস্টেম আপডেট)

একজন লিনাক্স ইউজার হিসেবে, পিসি অন করার পর প্রথম কাজ হলো টার্মিনাল খুলে সিস্টেম আপডেট করা। Windows এর মতো লিনাক্স নিজে থেকে বিরক্তিকর আপডেট রিস্টার্ট নেয় না।

### ডেইলি আপডেটের শর্টকাট (এগুলো রোজ করবেন):

```bash
# ১. রিপোজিটরি (সফটওয়্যার লিস্ট) আপডেট
sudo apt update

# ২. সফটওয়্যারগুলো ডাউনলোড ও ইন্সটল (-y মানে বারবার yes জিজ্ঞেস করবে না)
sudo apt upgrade -y
```

### 💡 প্রো-টিপ (Aliasing):

প্রতিদিন এত বড় কমান্ড না লিখে `~/.bashrc`-তে একটি অ্যালিয়াস বানিয়ে রাখুন (Section 21 তে শিখেছি):

```bash
alias update='sudo apt update && sudo apt upgrade -y'
# এরপর থেকে শুধু update লিখলেই কাজ শেষ!
```

## 🧹 2. System Cleanup (পিসি হালকা রাখা)

সময়ের সাথে সাথে উবুন্টুতে অনেক অব্যবহৃত প্যাকেজ এবং ক্যাশ জমে যায়। মাসে অন্তত ১-২ বার এগুলো পরিষ্কার করা উচিত।

### ক. Autoremove (সবচেয়ে গুরুত্বপূর্ণ ক্লিনআপ):

আপনি যখন কোনো সফটওয়্যার ইন্সটল করেন, তার সাথে কিছু সাহায্যকারী ফাইলও (dependencies) ইন্সটল হয়। পরে সফটওয়্যার ডিলিট করলেও সাহায্যকারীরা থেকে যায়। এগুলো মুছতে:

```bash
sudo apt autoremove -y
```

### খ. APT Cache Clear:

ডাউনলোড করা `.deb` ফাইলগুলো ক্যাশে জমা থাকে। স্পেস বাঁচাতে:

```bash
# শুধু অপ্রয়োজনীয় ক্যাশ মুছবে
sudo apt autoclean

# একদম সব ক্যাশ মুছবে (স্টোরেজ ফুল হলে இது করবেন)
sudo apt clean
```

### গ. Snap Cache Clear:

Ubuntu তে অনেক সফটওয়্যার Snap ফরম্যাটে থাকে, যা প্রচুর স্টোরেজ খায় পুরানো ভার্সন জমিয়ে রেখে।

```bash
# (এটি একটি Advanced script যা পুরানো স্ন্যাপ ডিলিট করে)
sudo bash -c 'set -eu; snap list --all | awk "/disabled/{print \$1, \$3}" |
    while read snapname revision; do
        snap remove "$snapname" --revision="$revision"
    done'
```

## 🌡️ 3. Hardware Health & Monitoring

আপনার প্রসেসর কি খুব বেশি গরম হচ্ছে? ব্যাটারির হেলথ কেমন? লিনাক্সে এগুলো চেক করা খুব মজার!

### ব্যাটারি মনিটরিং (`acpi`):

ল্যাপটপ ইউজারদের জন্য এটি খুব কাজের।

```bash
# ইন্সটল করে নিন
sudo apt install acpi -y

# ব্যাটারির বর্তমান অবস্থা ও চার্জিং টাইম
acpi

# ব্যাটারি ফুল ডিটেইলস (হেলথ/ক্যাপাসিটি)
acpi -V
```

### সিস্টেম টেম্পারেচার (`sensors`):

অতিরিক্ত গেম খেললে বা ভারী কাজ করলে টেম্পারেচার দেখতে:

```bash
# ইন্সটল করুন lm-sensors
sudo apt install lm-sensors -y

# সেন্সর ডিটেক্ট করতে (সবগুলোতে ENTER/yes দিন)
sudo sensors-detect

# তাপমাত্রা দেখতে
sensors
```

## 🚀 4. Startup Application Management

পিসি অন করতে কি খুব সময় লাগছে? তারমানে ব্যাকগ্রাউন্ডে অনেক অ্যাপ নিজে থেকে চালু হয়ে যাচ্ছে।

উবুন্টুতে "Startup Applications" নামের একটি গ্রাফিক্যাল অ্যাপ আছে (Activities থেকে সার্চ করুন)। টার্মিনাল দিয়েও ব্যাকগ্রাউন্ড সার্ভিস বন্ধ করা যায়:

```bash
# সব সার্ভিস কত সময় নিচ্ছে পিসি অন হতে তা চেক করা
systemd-analyze blame | head -n 10

# অব্যবহৃত সার্ভিস স্টার্টআপ থেকে বন্ধ করা (যেমন apache বা mysql যদি দরকার না হয়)
sudo systemctl disable apache2
```
