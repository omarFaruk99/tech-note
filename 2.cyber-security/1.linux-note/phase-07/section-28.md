## 🖥️ 1. Essential Desktop Workflow (বেসিক নেভিগেশন)

Ubuntu এর ডিফল্ট ডেস্কটপ এনভায়রনমেন্ট হলো **GNOME**। এটি Windows এর মতো নয়, বরং অনেক বেশি কীবোর্ড-নির্ভর হওয়ার জন্য ডিজাইন করা হয়েছে।

### 🚀 অ্যাপ ওপেন করার সবচেয়ে দ্রুত উপায়:

মাউস দিয়ে মেনুতে গিয়ে অ্যাপ খোঁজার চেয়ে:

1. শুধু `Super` (উইন্ডোজ) কী চাপুন।
2. অ্যাপের নামের প্রথম কয়েকটি অক্ষর (যেমন: `fi` Firefox এর জন্য বা `na` Nautilus/Files এর জন্য) টাইপ করুন।
3. `Enter` চাপুন। ব্যাস!

### 🗂️ ফাইল ম্যানেজার (Nautilus) শর্টকাট:

- **Ctrl + H:** লুকানো (Hidden) ফাইল ও ফোল্ডারগুলো দেখা বা লুকানো (যেগুলোর নামের আগে `.` আছে)।
- **Ctrl + L:** লোকেশন বার (Address bar) সিলেক্ট করা (যাতে আপনি সরাসরি পাথ টাইপ করে পেস্ট করতে পারেন)।
- **Ctrl + Shift + N:** নতুন ফোল্ডার তৈরি করা।
- **F2:** সিলেক্ট করা ফাইলের নাম রিনেম (Rename) করা।

## 🖼️ 2. Window Management (উইন্ডো সাজানো)

মাল্টিটাস্কিং বা কোডিং করার সময় স্ক্রিন শেয়ার করে দুটি উইন্ডো পাশাপাশি রাখাটা খুবই কমন।

### Split Screen (স্ক্রিন অর্ধেক করা):

- **Super + Left Arrow (←):** বর্তমান উইন্ডোটি স্ক্রিনের ঠিক বাম অর্ধেক জুড়ে বসবে।
- **Super + Right Arrow (→):** উইন্ডোটি ডান অর্ধেকের মাপে ফিট হয়ে যাবে।
  _(এটি কোডিং করার সময় একপাশে টার্মিনাল ও অন্যপাশে ব্রাউজার রাখার জন্য দারুণ!)_

### উইন্ডো কন্ট্রোল:

- **Super + Up Arrow (↑):** উইন্ডোটি ম্যাক্সিমাইজ (ফুল স্ক্রিন) হবে।
- **Super + Down Arrow (↓):** ম্যাক্সিমাইজ থাকলে ছোট হবে, আর ছোট থাকলে মিনিমাইজ (হাইড) হয়ে যাবে।
- **Super + D:** সব উইন্ডো হাইড করে সরাসরি ডেস্কটপ দেখা (Show Desktop)। दोबारा চাপলে আবার আগের অবস্থায় ফিরবে।
- **Alt + F4:** বর্তমান উইন্ডোটি ক্লোজ (বন্ধ) করা।

## 🚀 3. Workspaces (ভার্চুয়াল ডেস্কটপ)

Workspaces হলো লিনাক্সের অন্যতম সেরা ফিচার। এটি দিয়ে আপনি একাধিক "ডেস্কটপ" তৈরি করতে পারেন। যেমন: ওয়ার্কস্পেস ১-এ শুধু কোড এডিটর, ওয়ার্কস্পেস ২-এ ব্রাউজার, আর ওয়ার্কস্পেস ৩-এ Spotify।

### ওয়ার্কস্পেস ম্যানেজ করা:

- **Super + Page Up / Page Down (বা স্ক্রল):** ওয়ার্কস্পেসগুলোর মধ্যে ওপরে-নিচে সুইচ করা।
- **Super + Shift + Page Up/Down:** বর্তমানে যে উইন্ডোটি খোলা আছে, সেটিকে এক ওয়ার্কস্পেস থেকে অন্য ওয়ার্কস্পেসে পাঠানো।

## 📸 4. Screenshots & Recording (উবুন্টু ডিফল্ট টুল)

Ubuntu 22.04 এর পর থেকে স্ক্রিনশট এবং স্ক্রিন রেকর্ডিং এর জন্য একটি চমৎকার নতুন UI যুক্ত করা হয়েছে। এর জন্য কোনো থার্ড-পার্টি অ্যাপ লাগে না।

### নতুন স্ক্রিনশট টুল ওপেন করা:

- শুধুমাত্র **Print Screen (PrtScn)** কী চাপুন।
  _(এটি স্ক্রিনের নিচে একটি প্যানেল খুলবে, যেখান থেকে আপনি পুরো স্ক্রিন, একটি নির্দিষ্ট উইন্ডো, বা নির্দিষ্ট অংশ ক্রপ করে স্ক্রিনশট নিতে পারবেন। পাশেই ভিডিও আইকনে ক্লিক করলে স্ক্রিন রেকর্ড শুরু হবে।)_

### দ্রুত (Quick) শর্টকাট:

- **Shift + PrtScn:** সরাসরি সিলেকশন টুল আসবে (অংশবিশেষের স্ক্রিনশট নিতে)।
- **Alt + PrtScn:** শুধুমাত্র বর্তমানে চালু (Active) উইন্ডোটির স্ক্রিনশট নেয়া হবে।
- **Shift + Ctrl + Alt + R:** সরাসরি স্ক্রিন রেকর্ডিং শুরু এবং বন্ধ করা (স্ক্রিনের ডান একদম উপরে একটি লাল ডট দেখাবে)।

## 🔒 5. System Controls (সিস্টেম লক ও পাওয়ার)

অফিসে বা অন্য কোথাও ল্যাপটপ রেখে যাওয়ার আগে দ্রুত লক করার অভ্যাসটি সিকিউরিটির জন্য জরুরি।

- **Super + L:** স্ক্রিন লক (Lock) করা।
- **Ctrl + Alt + Delete:** লগআউট বা পাওয়ার অফ ডায়ালগ বক্স আনা।
- **Super + V:** নোটিফিকেশন প্যানেল ও ক্যালেন্ডার দেখা।

## 🏋️ Practice Exercise (Make it a Habit)

এই ব্যায়ামগুলো প্রতিদিন পিসি অন করে ২-৩ বার প্র্যাকটিস করুন, যতক্ষণ না এগুলো আপনার "Muscle Memory" তে সেট হয়ে যায়:

1. `Ctrl + Alt + T` চেপে টার্মিনাল খুলুন।
2. `Super + Left Arrow` চেপে টার্মিনালটিকে স্ক্রিনের বাম পাশে আটকে দিন।
3. `Super` চেপে অকারণে `firefox` টাইপ করে এন্টার দিয়ে ব্রাউজার খুলুন।
4. `Super + Right Arrow` চেপে ব্রাউজারটিকে ডান পাশে আনুন।
5. `Shift + PrtScn` চেপে এই সুন্দর সেটআপটার একটা স্ক্রিনশট নিন।
6. সবশেষে `Alt + F4` চেপে দুটি উইন্ডোই বন্ধ করে দিন।
