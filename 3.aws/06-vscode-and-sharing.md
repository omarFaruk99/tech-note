## 🔑 পার্ট ১: `.pem` ফাইল সেটআপ (অবশ্যই করতে হবে)

সার্ভারে ঢোকার আগে আপনাকে আপনার কম্পিউটারের টার্মিনালে চাবিটি সেটআপ করতে হবে। যেহেতু আপনি অফিস এবং বাসা দুই জায়গায় কাজ করেন, তাই আপনার জন্য দুটি আলাদা নিয়ম আছে।

### দৃশ্যপট এ: অফিস (Windows WSL ব্যবহার করলে)

আপনার অফিসের পিসিতে যদি Windows থাকে এবং আপনি **WSL (Ubuntu)** টার্মিনাল ব্যবহার করেন:

```bash
# ১. উবুন্টুর ভেতরে .ssh ফোল্ডার তৈরি করুন
mkdir -p ~/.ssh

# ২. Windows-এর Downloads থেকে ফাইলটি উবুন্টুতে নিয়ে আসুন
# (আপনার পিসিতে পাথ হলো: /mnt/host/c/Users/saaqe/Downloads/)
cp /mnt/host/c/Users/saaqe/Downloads/my-react-server-key.pem ~/.ssh/

# ৩. ফাইলের পারমিশন ঠিক করুন
chmod 400 ~/.ssh/my-react-server-key.pem
```

### দৃশ্যপট বি: বাসা (সরাসরি Ubuntu ব্যবহার করলে)

আপনার বাসার পিসিতে যদি সরাসরি **Ubuntu OS** থাকে:

```bash
# ১. .ssh ফোল্ডার তৈরি করুন
mkdir -p ~/.ssh

# ২. সরাসরি Downloads ফোল্ডার থেকে ফাইলটি কপি করুন
cp ~/Downloads/my-react-server-key.pem ~/.ssh/

# ৩. ফাইলের পারমিশন ঠিক করুন
chmod 400 ~/.ssh/my-react-server-key.pem
```

## 💻 পার্ট ২: VS Code দিয়ে কানেক্ট করুন

আপনার সব কম্পিউটারে (অফিস বা বাসা) এই নিয়মটি একই থাকবে:

### ধাপ ৬.১ — Extension ইনস্টল করুন

১. VS Code খুলুন।
২. বাম পাশের **Extensions** আইকনে ক্লিক করুন।
৩. সার্চ করুন: `Remote - SSH` এবং এটি ইনস্টল করুন।

### ধাপ ৬.২ — SSH Config ফাইল সেটআপ করুন

১. কিবোর্ডে `F1` বা `Ctrl + Shift + P` চাপুন।
২. লিখুন: `Remote-SSH: Open SSH Configuration File...`
৩. নিচের কোডটি সেখানে পেস্ট করুন:

```text
Host my-aws-server
    HostName YOUR_SERVER_IP (যেমন: 54.169.XXX.XXX)
    User ubuntu
    IdentityFile ~/.ssh/my-react-server-key.pem
```

## 🤝 পার্ট ৩: অন্যদের (যেমন সিনিয়রকে) এক্সেস দিন

### পদ্ধতি ১: .pem ফাইল শেয়ার করা

আপনার `my-react-server-key.pem` ফাইলটি তাকে দিতে পারেন। তিনি আপনার IP এবং এই ফাইলটি ব্যবহার করে কানেক্ট করতে পারবেন।

### পদ্ধতি ২: তার নিজের SSH Key যুক্ত করা (মডার্ন পদ্ধতি)

১. সিনিয়রের কাছ থেকে তার **Public Key** চেয়ে নিন।
২. আপনার সার্ভারে ঢুকে এই কমান্ডটি দিন:

```bash
nano ~/.ssh/authorized_keys
```

৩. ফাইলের একদম নীচে তার কি-টি পেস্ট করে সেভ করুন। এরপর তিনি তার নিজের পিসি থেকে পাসওয়ার্ড ছাড়াই আপনার সার্ভারে ঢুকতে পারবেন।
